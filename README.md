# AI Trivia Master 🧠🎯

AI Trivia Master is an interactive web application that allows users to test their knowledge through engaging trivia quizzes. The platform generates and presents quiz questions across different topics, providing users with an enjoyable way to learn and challenge themselves. Trivia and quiz-based systems are widely used to improve learning and knowledge assessment because quizzes help reinforce understanding and retention of information. ([arXiv][1])

This project demonstrates how modern web technologies can be used to build a dynamic quiz platform with a clean user interface and responsive experience.

## 🚀 Features

* Interactive trivia quiz gameplay
* Multiple-choice questions
* Instant feedback and score tracking
* Simple and user-friendly interface
* Responsive design for desktop and mobile
* Lightweight and fast web application

## 🛠️ Tech Stack

## ⚙️ Installation and Setup

1. Clone the repository
   git clone https://github.com/Hemanthkumar25s/AI-Trivia-Master.git

2. Navigate to the project folder
   cd AI-Trivia-Master

3. Open the project
   Open the **index.html** file in your browser to run the application.

## 💡 How It Works

Users start the quiz and answer multiple-choice questions presented by the application. The system evaluates the answers instantly and displays the score or feedback, making the quiz interactive and engaging.
